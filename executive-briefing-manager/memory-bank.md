## Key Connector Operations

- `Office365UsersService.MyProfile_V2($select)` → logged-in user name, title, email, id
- `Office365UsersService.UserPhoto_V2(id)` → photo (multi-format: base64 string OR `{$content, $content-type}` object)
- `Office365UsersService.SearchUserV2(searchTerm, top)` → tenant user search
- `Office365OutlookService.GetEventsCalendarViewV3('Calendar', startISO, endISO)` → calendar events for date range

## Known Issues / Gotchas

- `IOperationResult<T>` shape: `{ success: boolean, data: T }` — NOT `{ isSuccess, value }`
- `UserPhoto_V2` returns photo in multiple formats — use `extractPhotoDataUrl()` helper in `userService.ts`
- Vite auto-incremented to port 3001 (3000 was in use); `pac code run` needs `--appUrl http://localhost:3001`
- Generated files in `src/generated/` — never edit manually
- **Blank screen after rapid edits**: Vite HMR cache gets stale after multiple rapid changes. Fix: restart Vite (`stop_powershell vite-devX` then restart). Hard refresh (`Ctrl+Shift+R`) also helps.
