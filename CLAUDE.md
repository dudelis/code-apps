# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository purpose

A collection of standalone web apps and learning materials for Claude Code. Each app lives in its own subdirectory and is a self-contained `index.html` (no build step, no package manager).

## Running apps

Open any `index.html` directly in a browser — there is no server or build process required.

## Structure

- `tictactoe/` — browser-based Tic Tac Toe game (single HTML file with embedded CSS + JS)
- `Preparation/` — notes and installation guides for Claude Code
